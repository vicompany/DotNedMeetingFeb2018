Missende features; Data uitlezen uit externe bronnen, geen random, wel throw maar geen catch
Extra features;
- calls (constant), transactions en payable transactions
- restrictie van gasverbruik
- function modifiers
- msg object

Ontwikkelen van smart contract

						contract HelloWorld { 
    string lastGreeting;
    
    function HelloWorld() public {
     
    }
    function SetGreeting(string greeting)  {
         lastGreeting = greeting;
    }
    
    function Greet() constant returns (string) {
         return lastGreeting;
    }
    
}
